# Project homework_11.2

## Project homework_11.2 это веб-приложение на Python для сортировки списков словарей по значению для ключа state и по дате(времени), формирования масок номеров карт и счетов, корректного отображения временных данных в формате ДД.ММ.ГГ.

## Для установки клонируйте репозиторий:
```
git clone https://github.com/AlexChapov/homework_10.1.git
```
## Функция filter_by_state принимает на вход список словарей и возвращает сортированные списки по значению для ключа state (EXECUTED и CANCELED)

## Функция sort_by_date принимает на вход список словарей и возвращает сортированные списки по дате/времени (в порядке убывания)

## Функция get_mask_card_number создающая маску для номера карты.

## Функция get_mask_account создающая маску для номера счета.

## Функция mask_account_card создающая маску для полученных данных карт, счета

## Функция get_date создающая дату в формате ДД.ММ.ГГГГ из данных полученных в формате 2024-03-11T02:26:18.671407.

## Функция filter_by_currency принимает на вход список словарей, представляющих транзакции, возвращает транзакции, где валюта операции соответствует заданной (['code'] == 'USD')

## Функция transaction_descriptions принимает список словарей с банковскими операциями и возвращает описание каждой операции по очереди.
